# 개발자 웹 이력서 프로젝트 로드맵

## 프로젝트 개요
개인 포트폴리오 및 이력서를 소개하는 현대적이고 반응형 웹사이트 개발

---

## 기술 스택
- **마크업**: HTML5
- **스타일링**: CSS3, Tailwind CSS
- **인터랙션**: JavaScript (Vanilla)
- **배포**: GitHub Pages / Netlify (선택 사항)

---

## 개발 단계

### Phase 1: 프로젝트 구조 설정 ✅ (준비)
- [ ] 프로젝트 디렉토리 생성
- [ ] HTML, CSS, JavaScript 파일 구조 설계
- [ ] Tailwind CSS 설정 (CDN 또는 npm)
- [ ] `.gitignore`, `README.md` 작성

### Phase 2: 레이아웃 개발
- [ ] 헤더 / 네비게이션 바 (반응형)
- [ ] 히어로 섹션 (자기소개)
- [ ] 약력 섹션 (경력, 교육)
- [ ] 기술 스택 섹션
- [ ] 프로젝트 포트폴리오 섹션
- [ ] 연락처 / 풋터

### Phase 3: 스타일링 및 디자인
- [ ] Tailwind CSS로 컴포넌트 스타일링
- [ ] 색상 팔레트 정의 (일관성 유지)
- [ ] 타이포그래피 설정
- [ ] 반응형 디자인 (모바일, 태블릿, 데스크톱)
- [ ] 다크 모드 지원 (선택 사항)

### Phase 4: 인터랙션 추가
- [ ] 부드러운 스크롤 네비게이션
- [ ] 모바일 메뉴 토글 기능
- [ ] 이미지 Lazy Loading
- [ ] 호버 애니메이션 효과
- [ ] 스크롤 애니메이션 (AOS 또는 Intersection Observer)

### Phase 5: 기능 구현
- [ ] 연락처 폼 (이메일 연동 - FormSubmit 등)
- [ ] 다운로드 버튼 (PDF 이력서)
- [ ] 소셜 미디어 링크
- [ ] 검색 최적화 (SEO)
- [ ] 성능 최적화

### Phase 6: 테스트 및 배포
- [ ] 크로스 브라우저 호환성 테스트
- [ ] 모바일 반응형 테스트
- [ ] 접근성(A11y) 검토
- [ ] 성능 체크 (Lighthouse)
- [ ] GitHub Pages 또는 Netlify 배포

## 이력서 콘텐츠 구조

### 1. 개인정보
- 이름, 직책
- 프로필 이미지
- 짧은 소개 (2-3줄)

### 2. 핵심 역량
- 주요 기술 스택
- 전문 분야 (3-5개)

### 3. 경력사항
- 회사명 / 프로젝트명
- 기간
- 담당 역할 및 성과 (2-3줄)

### 4. 기술 스택
- 프로그래밍 언어
- 프레임워크 / 라이브러리
- 도구 및 플랫폼

### 5. 프로젝트
- 프로젝트 이름
- 설명 (1-2줄)
- 사용 기술
- GitHub 링크 / 라이브 데모

### 6. 교육 및 자격증
- 학력
- 취득 자격증 / 수료증

### 7. 연락처
- 이메일
- GitHub
- LinkedIn
- 포트폴리오 사이트
